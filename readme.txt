Zeigt Statistiken von www.counterfire.info im WordPress Dashboard.

== Description ==

Counterfire.info bietet kostenlose Statistiken. Mit diesen Plugin kann der von Counterfire bereitgestellte Besucherzähle automatisch eingebunden werden und die Statistik wird direkt im Dashboard angezeigt.

Weitere Informationen und Plugins auf der Pluginseite: [http://sxss.nw.am](http://sxss.nw.am "sxss.nw.am")

== Installation ==

zip Datei entpacken und den Inhalt in den Ordner /wp-content/plugins/ schieben, aktivieren.
